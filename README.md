# 🚀 Sarujan Marianesan — Portfolio

A modern, responsive personal portfolio website showcasing my journey as a **Software Engineering Undergraduate** at the University of Westminster.

## 🌐 Live Demo

🔗 [View Portfolio](https://saru59427-debug.github.io/Portfolio/)

## ✨ Features

- **Glassmorphism UI** — Premium dark-themed design with frosted glass cards and glowing accents
- **Fully Responsive** — Optimized for desktop, tablet, and mobile devices
- **Animated Interactions** — Smooth AOS scroll animations, typed text effects, and hover micro-animations
- **Multi-Page Layout** — Dedicated pages for Home, About, Education, Skills, Projects, Experience, and Contact
- **Contact Form** — Functional contact form powered by Formspree
- **Direct Social Links** — LinkedIn, GitHub, Gmail compose, and WhatsApp integration
- **SEO Optimized** — Proper meta tags, Open Graph support, and semantic HTML

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/saru59427-debug/Portfolio.git
   ```

2. **Open locally**
   ```bash
   cd Portfolio
   ```
   Open `index.html` in your browser.

3. **Deploy**
   Push to GitHub and enable GitHub Pages under **Settings → Pages → Deploy from branch (main)**.
